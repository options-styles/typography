Typography

The typography consists of two type-faces, Colfax and Adelle Sans, used for display and body copy respectively.

Colfax is used as a display font as it's form fits well with the current brand typeface. In general we say that any text about 20px should be set in Colfax, and anything below should be set in Adelle. The font sizes used have been chosen specifically for their flexibility and their harmonious composition together. No other title sizes should be created without a good reason for doing so. The type scale is built on a ratio of 1.618 starting from a base size of 14px.

Usage Guidelines

  • Base font-size of 14px
  • Baseline is 20px
  • Don't create any new heading sizes
  • Use Colfax for any type over 20px
  • Use Colfax for any all-caps text
  • Use Colfax for interface elements
  • Type sizes are in ems, relative to the base font-size

Basic Usage

The copy to the left showcases all of the type styles available within the .t-copy module. Lists sit outside of the content for easier reading but can be pulled inside if needed.

Use the class .t-copy on a block to get the typography styles. All this code is contained within the typography CSS component

Classes are not needed to set type styles when they are within the content block, the elements are styled automatically.
